## Cервис одноразовых ссылок
----------

### Использование сервиса (dev режим):
### 1. Подготовка

1. Добавьте `.env` файл в соответствии с `.env.example`.
2. Установка зависимостей:
`npm i`

### 2. Запуск
`npm run start:dev`

### 3.Использование
Эндпойнт для генерации ссылки: `POST /one-time-link`
